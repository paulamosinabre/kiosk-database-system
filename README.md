# kiosk-database-system
This repository is intended for managing the database for the kiosk system entitled "Karinderya ni Aling Bebang," mainly for educational purposes as the final project for the course CTFBMSL, or Database Management, at National University - Dasmarinas.

This project involves an ordering and queue management system designed for both customers and administrators. It allows customers to browse food items, place orders, manage carts, and generate receipts, while administrators can manage products, monitor queues, and generate sales reports.

###### This system has the following tech stack:
- Java
- Java Swing
- MySQL
